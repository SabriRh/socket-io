This is an exemple of using socket.io for real time comunication between client and server. Our server is Node.js server using Express.js. The client is a simple Angular Application.
this App will display a welcome message from the server and the server will trigger a sample function that simply send the current time evry 10sec.
The page controllers will store them in $scope and display them on the Page Using data-binding, Cool !
